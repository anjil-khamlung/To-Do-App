# To-Do App (JavaScript)

A simple and responsive To-Do application built using HTML, CSS, and JavaScript.

## Features
- Add new tasks
- Mark tasks as completed using checkbox
- Delete tasks
- Tasks are saved using localStorage
- Responsive design without media queries

## Technologies Used
- HTML
- CSS (Flexbox)
- JavaScript (DOM, Events, localStorage)

## How to Run
1. Clone the repository
2. Open `index.html` in a browser

## Purpose
This project was built to strengthen JavaScript fundamentals before learning React.
